Basic Set up of GitHub Repository

  1. Create a GitHub Account
    • Visit https://github.com and sign up.

  2. Create a New Repository
    • Log into GitHub and click the "+" icon in the top-right corner
    • Select "New repository"
    • Fill in the repository details like: Repository name, Add a description (optional), Choose Public or Private, Check Initialize this repository with a README (Optional)
    • Click Create repository
